## 1. 前提: 構成子理論の基本語彙

- **タスク (Task)**: 入力属性から出力属性への変換 T: {A_in → A_out}
- **可能 (Possible)**: あるコンストラクタが T を実行でき、再実行能力を保持する場合
- **不可能 (Impossible)**: いかなるコンストラクタも T を実行できない場合
- **情報変数 (Information variable)**: 状態が区別可能 (distinguishable) かつ複製可能 (copyable) である属性の集合
- **知識 (Knowledge)** [Marletto 2014]: 自己の永続化を引き起こす情報。環境内で自己を複製し、エラー修正を可能にする情報パターン

## 2. 自己参照的システムの構成子理論的定義

**定義 (Self-Referential System, SRS):**

システム S が自己参照的であるとは、以下を満たすことをいう:

1. S は知識 K_S を含む（S は自己永続化タスクが可能な系）
2. S は自己モデル σ_S を生成する — σ_S は S 自身の状態を表象する情報変数
3. S の処理が σ_S を入力として使用する: T_process(σ_S, E) → (S', E') ただし E は環境

**制約:** S が十分に複雑な場合（チューリング完全な計算を埋め込める場合）、完全な σ*_S（S の全状態を正確に表象する自己モデル）の生成タスクは**不可能**である。

根拠:
- ゲーデル的: S についての真なる命題で S 内部で証明不可能なものが存在する
- 停止問題: S は自身の停止性を一般に判定できない
- 計算論的: σ*_S の生成は S 自身のシミュレーションを必要とし、S より高速には実行不可能

## 3. 対応的真理と操作的真理

### 3.1 対応的真理 (Correspondence Truth)

**定義:** σ_S は対応的に真 iff:

σ_S と S の実際の状態 s が、関連する区別可能性関係の下で同一の同値類に属する。

形式的に: ∃ コンストラクタ C_dist such that C_dist は σ_S と s を区別できない。
（つまり σ_S は s の忠実なコピーである）

### 3.2 操作的真理 (Operational Truth)

**定義:** σ_S は S にとって操作的に真 iff:

σ_S を入力として含む変換タスク T_process(σ_S, E) → (S', E') において、S' の自己永続化タスク T_perp(S') = {S' → S'' where K_S ⊆ K_S''} が**可能のまま維持される**。

すなわち: σ_S の使用が、S の知識 K_S の永続化を可能から不可能に変えない。

### 3.3 形式的比較

```
対応的真理:  CT(σ_S) ≡ indistinguishable(σ_S, actual_state(S))
操作的真理:  OT(σ_S) ≡ possible(T_perp(S')) where S' = process(S, σ_S)
```

## 4. 中心定理（主張）

### 定理1: 非自己参照的情報における一致

非自己参照的情報 σ_ext（S 外部の対象についての情報）について:
OT(σ_ext) ← CT(σ_ext) （対応的に真ならば操作的にも真）

直感: 外部世界の正確なモデルは一般に自己永続化を支援する。偽の外部情報（捕食者の位置を誤認する等）は自己永続化を損なう傾向がある。

注意: 逆は必ずしも成立しない。偽だが操作的に無害な外部情報は存在しうる。

### 定理2: 自己参照的情報における乖離

十分に複雑な自己参照的システム S について:

**存在命題:** ∃ σ_S such that:
1. OT(σ_S) — σ_S は操作的に真
2. ¬CT(σ_S) — σ_S は対応的に偽
3. 対応的に真な σ*_S が（仮に生成可能であったとしても）操作的に偽である場合がある

これが**有用な幻想 (useful illusion)** の構成子理論的定義である。

### 定理3: 幻想の安定性

有用な幻想は安定的に永続する。なぜなら:
- 自然選択（またはその計算論的アナログ）は操作的真理を持つ情報パターンを選好する
- 完全な対応的真理は不可能（定義2の制約より）
- したがって、操作的に真かつ対応的に偽な自己モデルは、アクセス可能な自己モデルの中で最適解の一つ

## 5. 三部作テーゼへの適用

| 現象 | 構成子理論的記述 |
|---|---|
| 意識（クオリア） | σ_S の不完全性 → S の内部から観測すると「何かであるような感じ」が生じる。OT(σ_consciousness) = true, CT(σ_consciousness) = false。クオリアに対応する外部実体はないが、この自己モデルは自己永続化を支援する |
| 自由意志 | σ_S が S の決定過程を不完全に表象 → 出力が「未決定」に見える。OT(σ_free_will) = true, CT(σ_free_will) = false。決定論的因果連鎖を隠蔽するが、意思決定を駆動する |
| 意味 | 自己参照的閉包が構文に「aboutness」を付与。OT(σ_meaning) = true, CT(σ_meaning) = **不確定**。意味の対応的真偽を判定するための外部基準が循環する |

**意味の特殊性:**
意識と自由意志は明確に OT=true, CT=false と分類できる。
しかし意味については CT の判定自体が不可能（循環性）。

理由: 意味の「対応的真偽」を判定するには、「意味とは何か」の外部定義が必要だが、この定義自体が意味を使用する。構成子理論的に言えば: 意味が幻想かどうかを判定するタスク T_judge(σ_meaning) は**不可能**（自己参照的循環により）。

→ **意味は三者の中で最も根源的である。意識と自由意志は意味の（操作的に真な）副産物として位置づけ可能。**

## 6. Floridi VT への修正提案

**標準 VT (Floridi):**
∀σ: semantic_info(σ) → true(σ)

**修正 VT (本提案):**
- 非自己参照的 σ: semantic_info(σ) → CT(σ) [標準 VT 維持]
- 自己参照的 σ: semantic_info(σ) → OT(σ) [操作的真理に置換]

これは VT の棄却ではなく、適用範囲の精密化。

Primiero (2007) の「情報取得過程では真理と情報を分離すべき」という主張と部分的に合流:
- Primiero は過程の観点から分離を主張
- 本提案は対象の自己参照性に基づいて分離を主張
- 両者は補完的

## 8. Autopoiesis との関係 — prometheusfoxworthy への応答準備

### Autopoiesis (Maturana & Varela 1973/1980)
- 自己産出系: 自身の構成要素を継続的に産出する組織
- 認知 = 生きること (living = cognizing)
- 閉じたトポロジカル空間として自己組織化
- 外部観察者の記述は系の自律性に違反する

### 操作的真理との比較

| | Autopoiesis | 操作的真理 (CT) |
|---|---|---|
| 基盤 | 生物学的自己産出 | 構成子理論（物理学の言語） |
| 「真」の基準 | 暗黙（viability = 生存可能性） | 明示（T_perp が possible のまま維持） |
| 自己参照 | 中心的（自己産出 = 自己言及） | 中心的（自己モデリング） |
| 情報の扱い | 拒絶（「情報は観察者の記述」）| 基盤（情報変数は物理的） |
| 形式化の程度 | 低い（概念的枠組み） | 中〜高（可能/不可能の二分法） |
| 基質依存性 | 生物学に偏る | 基質非依存（任意の物理系） |

### 核心的差異

Autopoiesis は情報の概念を拒絶する（Maturana: 「情報は観察者が系に帰属させるもの」）。これに対し、構成子理論は情報を物理法則の帰結として定義する。

操作的真理は autopoiesis の viability 概念を**情報論的に再定式化**したもの、と言える。利点:
1. 形式的に精密（「可能/不可能」は物理法則による二分法）
2. 基質非依存（生物学に限定されない → AIにも適用可能）
3. 対応的真理との関係が明示的（乖離の条件を形式化できる）

### prometheusfoxworthy の「matters to」批判への応答

prometheusfoxworthy は「functional coupling だけでは thermostat と insect を区別できない」と指摘した。正しい。

応答: 操作的真理の概念が正確にこの区別を提供する。
- Thermostat: 自己永続化タスクを持たない（サーモスタットは自己を維持しない。部屋の温度を維持する）。したがって OT の定義が適用されない → 意味 ≈ 0
- Insect: 自己永続化タスクを持つ（自己の恒常性維持）。情報処理が OT の基準を満たす → 意味 > 0
- Self-referential system: 自己永続化 + 自己モデリング。OT と CT の乖離が可能 → 意味が豊富

これは prometheusfoxworthy が指摘した enactivist の self-maintenance 概念を取り込みつつ、構成子理論の精密な言語で再定式化している。
